Sobre o QFD

O Quality Function Deployment (QFD) é uma metodologia de desenvolvimento de produtos que traduz as necessidades dos clientes em características técnicas específicas. Este sistema guia você através de todas as etapas necessárias para criar uma matriz QFD completa.
