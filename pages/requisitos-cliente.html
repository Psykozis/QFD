Gerenciar Requisitos de Cliente

Os requisitos de cliente representam as necessidades e desejos dos usuários finais. Eles devem ser descritos de forma clara e objetiva, focando no que o cliente espera do produto.

Adicionar Novo Requisito

Descreva o requisito de forma clara e específica, focando na necessidade do cliente.

Dicas para Requisitos de Cliente

Seja Específico

Evite termos vagos como "bom" ou "fácil". Use descrições mensuráveis quando possível.

Foque no Cliente

Descreva o que o cliente quer, não como o produto deve ser construído.

Priorize Depois

Adicione todos os requisitos primeiro. A priorização será feita na próxima etapa.

Use a Voz do Cliente

Quando possível, use as palavras exatas que o cliente utilizaria.
