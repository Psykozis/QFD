Gerenciar Requisitos de Projeto

Os requisitos de projeto são características técnicas mensuráveis que definem como o produto será construído para atender às necessidades dos clientes. Cada requisito deve ter um sentido de melhoria claramente definido.

Adicionar Novo Requisito de Projeto

Descreva uma característica técnica mensurável do produto. Pressione Enter para salvar.
Indique se valores maiores, menores ou um valor específico representam melhoria.
Avalie a dificuldade de implementação ou melhoria deste requisito.

Sentidos de Melhoria

↑

Crescente

Valores maiores representam melhoria

Ex: Velocidade, Capacidade, Eficiência
↓

Decrescente

Valores menores representam melhoria

Ex: Tempo de resposta, Peso, Custo
*

Nominal

Um valor específico é o ideal

Ex: Temperatura, pH, Dimensões

Dicas para Requisitos de Projeto

Seja Mensurável

Use características que podem ser medidas objetivamente com unidades específicas.

Foque na Técnica

Descreva como o produto funciona, não o que o cliente quer.

Defina a Direção

Sempre especifique se maior, menor ou um valor específico é melhor.

Considere a Implementação

Avalie realisticamente a dificuldade técnica de cada requisito.
